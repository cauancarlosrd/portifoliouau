Bem-vindo ao meu laboratório digital. Este domínio não usa frameworks mastigados. Tudo aqui é construído na unha, tijolo por tijolo, escavando o DOM e dobrando o CSS à minha vontade. O objetivo não é apenas ter um site, é entender as engrenagens por baixo do chassi.
